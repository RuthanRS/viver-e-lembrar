
  # Viver é Lembrar

  This is a code bundle for Viver é Lembrar. The original project is available at https://www.figma.com/design/GIOZkMnJujNVuYGieczeUg/Viver-%C3%A9-Lembrar.

<u><strong>Por que este projeto?</strong></u>


Embora já existam aplicativos voltados ao suporte de pessoas com Alzheimer, acredita-se que o tema ainda merece maior atenção dentro da tecnologia.

A doença afeta milhões de pessoas no mundo e impacta diretamente pacientes, familiares e cuidadores.
Projetos como o Viver é Lembrar buscam reforçar a importância de desenvolver soluções acessíveis que auxiliem na rotina diária, promovendo mais autonomia, segurança e bem-estar.

<u><strong>Este é um projeto pessoal desenvolvido para fins de estudo e exploração de soluções tecnológicas voltadas ao apoio de pessoas com Alzheimer e seus familiares.</strong></u>
